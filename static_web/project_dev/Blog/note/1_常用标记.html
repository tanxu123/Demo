1_常用标记

1.定义页面元信息标记<meta>

<!--设置网页的关键字-->
<meta name="keywords" content="java培训。。。.net培训。。。"/>
<!--设置网页描述-->
<meta name="destribution" content="IT培训的龙头老大。。口碑最好"/>
<!--设置网页作者-->
<meta name="author" content="方志月"/>
<!--设置服务器发送的给浏览器的http头部的信息-->
<meta http-equiv="Content-Type" content="text/html;charset=utf-8"/>
<!--设置页面自动刷新与跳转-->
<meta http-equiv="refresh" content="10;url=http://www.itcast.cn"/>

    5.css属性align的使用

<p align="center">这是一个段落</p>    

     align属性在很多标记都能使用，不仅仅是段落  ，一般是只能在块级元素里面使用，具体在行内块里使用下面进行测试

<!DOCTYPE html>    <!--界面使用的是标准的html5标准-->
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        * {   <!--这个的作用是清除标记的原有的margin和padding以及border，因为部分标记会影响我们对整个网页的布局的判断-->
            margin: 0px;
            padding: 0px;
            border: none;
        }
        li {   <!--这个我们设置li为行内元素，就会不在独占一行-->
            display: inline;
        }
    </style>
</head>
<div>
    <!---下面的标记使用的宽度的值使用的是%的比例值，为了三个元素占一行，我们使用的是行内块，
    没有使用行内元素的原因是行内元素不能使用text-align属性，这个和align属性是一样的-->
    <ul style="display: inline-block;width: 10%;">
        <li>点击</li>
        <li>注册</li>
    </ul>
    <ul style="display: inline-block;width:80%;text-align: center">
        <li>我们的实现基本的功能</li>
        <li>注册</li>
    </ul>
    <ul style="display: inline-block;width:9%;text-align: right">
        <li>点击</li>
        <li>注册</li>
    </ul>
</div>
</body>
</html>  <!--上面可以作为一个模板，用来实现一行三块元素的不同显示方向-->
