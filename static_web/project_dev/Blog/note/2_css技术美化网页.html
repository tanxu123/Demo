  2.css基础选择器

1.标记选择器

标记名{属性名1:属性值1;属性名2:属性值2;}  该语法中，所有的标记都能作为标记选择器  html ，body，p等

2.类选择器

.类名{属性名1:属性值1;属性名2:属性值2;}     记住类可以多个，一个标记可以使用多个类。中间用空格隔开

3.id选择器

#id名{属性名1:属性值1;属性名2:属性值2;}      id选择器是唯一的，不能出现多个

4.通配选择器

*{属性名1:属性值1;属性名2:属性值2;}   匹配页面的所有元素

5.css复合选择器

<!--标签指定式选择器,又被称作交际选择器   由两个选择器组成，第一个为标记选择器，第二个为class选择器或id选择器-->
<!--后代选择器就是把外标签写在前面，后面跟上内标签，可以多重嵌套-->
<!--并集选择器，使用上面的所有选择器，使用逗号分开就行-->
<head>
    <style>
    p.mycapter{  /*标签指定式选择器*/
        color:green;
    }
    h1#title{   /*标签指定式选择器*/
        color:blue;
    }
    body p img{  /*后代选择器*/
        width:300px;
    }
    p.mycapter,h1#title{ /*并集选择器*/
        text-align:center;
    }
    </style>
</head>
<body>
    <h1 id="title">标题</h1>
    <p class="mycapter">
    <img src="images/top.jpg"/>
    </p>
</body>

4.css的层叠性与继承性

1.层叠性：多种样式的叠加，

2.继承性：在书写css样式表时，子标记会继承父标记的某些样式，但是如果网页使用了大量的继承，那么判断的样式会很麻烦，所以对于字体，文本属性等网页的常用样式可以使用继承，但并不是所有的元素都可以继承，例如下面的元素

  1. margin
  2. border
  3. padding
  4. background
  5. position
  6. 布局属性
  7. 元素宽高属性

记住，我们如果为body设置属性，但是并不会影响到h1-h6的设置，原因是h1-h6的语义较强，代表的就是标题

 

3.css的优先级

        1.标记选择器的权重：1       类选择器权重：10       id选择器权重：100           

         对于通用选择器来说，id选择器具有最高的优先级别

        2.继承样式的权重为0，即子啊嵌套过程中，不论父标记的选择权重多大，子标记继承时，权重为0，也就是子元素定义的样式会覆盖继承的样式

strong{color:red;}
#header{color:green;}

    对应的html

<p id="header" class="blue">
    <strong>继承的样式不如自己定义</strong>
</p>

        3.行内式有限，应用style属性的元素，其行内样式的权重非常高，可以理解为远大于100，总之比上面提到的都大

        4.权重相同时，css遵循就近原则，也就是说靠近元素的样式具有最大的优先级，或者说排在最后的样式优先级最大

        5.css定义了一个!important命令，该命令被赋予了最大的优先级别

#header{
    color:red!important;   /*位于属性值与分号之间*/
}

        6.复合选择器的坑

  1. 类选择器的的权重大于后代选择器，不论后代选择器有多少个标记选择器进行叠加
  2. 复合选择器的权重无论多少个类选择器和标记选择器的叠加其权重都不会超过id选择器
