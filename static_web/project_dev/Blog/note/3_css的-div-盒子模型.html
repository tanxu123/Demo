1.margin居中的例子

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        * {
            margin: 0px;
            border: none;
            padding: 0px;
        }
    </style>
</head>
<body>
<div style="text-align: center;width: 500px;margin: 0 auto;">
    <h2>标题是什么？</h2>
</div>
</body>
</html>
<!--我们在div里面包括了一个h2，当我们设置div的宽时，这时div将居左，尽管里面的元素是对齐的，所以我们可以通过margin:0 auto实现居中-->
那么我们可能会好奇，text-align与margin:0 auto 的使用区别在哪里
1.text-align:名字很形象，只能使文本居中显示，不能改变其他没有文本的标记的位置
2.margin:0 auto;:实现的就是元素的居中显示，当然仅仅限于块级元素

1.  块级元素

  1. 独自占一行
  2. 可以设置宽高，对齐属性
  3.  常用的元素有 <h1>~<h6>,<p>,<div>,<ul>,<ol>,<li>

2.行内元素

  1. 不必单独一行
  2. 不独占一个区域，仅仅依靠自身的字体大小以及图像的尺寸来支撑结构
  3. 一般不能设置宽高
  4. 常见元素  <strong>,<b>,<em>,<i>,<del>,<s>,<ins>,<u>,<a>,<span>

行内元素通常被包含在块级元素里面使用，但是行内元素不能嵌套块级元素

3.元素的转换

  1. 通过css的display可以实现元素类型的转换
  2. inline：将此元素显示为行内元素（默认值）
  3. block：显示为一个块级元素
  4. inline-block：显示为一个行内块元素，可以设置宽高和对齐方式，但是该元素不能不会独占一行
  5. none：元素被隐藏，不显示，也不用占用空间，相当于不存在
