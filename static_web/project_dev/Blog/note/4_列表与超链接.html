4_列表与超链接

 

1.无序列表

<ul>
    <li>列表1</li>
    <li>列表2</li>
</ul>
  1. ul和li都是块级元素，可以包含任意元素，都拥有一个type属性，用来指定列表项目符号 disc  circle  square
  2. 不赞成使用type属性，一般通过css进行替代
  3. <ul>只能嵌套<li>但是<li>可以嵌套任意元素

2.有序列表

<ol>
    <li>列表1</li>
    <li>列表2</li>
</ol>
  1. 除了type属性外，还可以为<li>和<ol>定义start，为<li>定义value

3.定义列表

<dl>
    <dt>名词</dt>
    <dd>名词解释</dd>
    <dd>名词解释</dd>
    ...
    <dt>名词</dt>
</dl>
  1. <dt>与<dd>并列嵌套
  2. <dt></dt>标记用于指定术语名词
  3. <dd>用于对名词进行解释，当然可以一个名词多个解释

4.list-style复合属性

list-style：列表项目符号  列表项目符号位置  列表项目图像
/*
在实际工程中，一般更好的实现效果，需要将list-style=none，然后通过li设置效果
*/

5.超链接

<a href="http://www.baidu.com" target="弹出窗口的方式" >文本或图像</a>
  1. href:指定链接地址
  2. target：_self    _blank
  3. 在没有确定链接目标时，通常将href属性设置“#” ，表示一个空链接
  4. 在网页中，图像，表格，音频，视频都能添加超链接

6.超链接伪类

伪类含义
a:link{ css样式规则 }未访问的状态
a:visited{ css样式规则 }访问后的状态
a:hover{ css样式规则}鼠标悬停，经过的状态，颜色
a:active{ css样式规则}单击不同的状态
  1. 定义伪类时 需要按照  a:link .a:visited a:hover a:active的顺序定义，否则会出现不能使用的效果

7.瞄点

  1.     使用<a href="#id名">链接文本</a>创建链接文本
  2. 使用相应的id名称标注跳转的地址
