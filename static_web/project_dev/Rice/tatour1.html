服务指南
1.1 自助服务

电信客户需要登录，才能进入到自助服务专区页面，并进行相关操作。

1.1.1 用户中心 1.1.1.1 功能说明

用户中心页面，主要展示用户信息，消费金额，帐户余额，我的套餐以及热门业务等信息。


1.1.1.2 具体操作

步骤一：打开江苏省电信网上营业厅主页http://189.cn/js/

步骤二：点击导航条上的“自助服务”；

步骤三：页面跳转自助服务专区登录页面，共有4中登录方式（手机，固话，宽带，其他）；

步骤四：选择一种登录方式（以手机为例），输入正确手机号码，以及手机密码后，点击登录；

步骤五：登录成功后，默认进入到用户中心页面；

步骤六：点击日期下的“《”链接；

进入到该商品的详情页面；

步骤七：选择登陆帐户下的其他帐户号码（当前帐户为客户级密码），点击标识“”；

包含的其他帐户号码信息；

步骤八：查看“我的套餐”页面，用户可以查看某套餐的消费情况；

步骤九：查看“热门业务”页面，可以进行“办理”与“详情”的操作；

业务“上网流量包”的详情页面信息；
